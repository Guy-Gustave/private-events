# Private Events

Users can create events. Events take place at a specific date and have description and they are grouped as upcoming and past.

## Built With

- Ruby 2.7.0
- Rails 6.0.3.2
- VS Code

## Prerequisites

In order to make the program work, you need to have ruby interpreter installed in your system. You can get the latest version of ruby from [official website](https://www.ruby-lang.org/en/downloads/).

# Getting Started

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <folder>
```

```bash
$ git clone git@github.com:Guy-Gustave/private-events.git
$ cd private-events
$ bundle install
$ yarn install --check-files
```
